Import Your Watchlist

Import your existing watch history from popular services. Export your data from each platform and upload the CSV file here.

How to export from Letterboxd:

  1. Go to letterboxd.com/settings/data/
  2. Click "Export Your Data"
  3. Download the ZIP file and extract it
  4. Upload one of these files:
    • watched.csv - All films you've watched
    • ratings.csv - Films with your ratings
    • watchlist.csv - Your watchlist
    • diary.csv - Your diary entries

How to export from IMDB:

  1. Go to your IMDB Ratings or Watchlist
  2. Click the three dots menu (⋮)
  3. Select "Export"
  4. Upload the downloaded CSV file

Custom CSV Format:

Your CSV should have headers. Supported columns:

  • title or name - Movie/series title (required)
  • year - Release year
  • type - "movie" or "series"
  • rating - Your rating (0-10)
  • status - "watched", "watching", or "want-to-watch"
  • genre or genres - Genre(s); use Games for video games
  • imdb_link, poster_url, letterboxd_link, rotten_tomatoes_link, justwatch_link, notes
  • imdb_rating, rt_rating - IMDB score and Rotten Tomatoes % (optional)

Use Settings → Export CSV to download this app’s lists in the same column layout.

🎬 TMDB API Key Recommended

Better posters, more metadata. Optional on deployed sites — only needed if you use this app without a server proxy, or to override the server keys in this browser.

How to get your free API key:

  1. Go to themoviedb.org/signup and create account
  2. Go to Settings → API
  3. Click "Create" → Select "Developer"
  4. Fill the form (use "Personal" for type)
  5. Copy your API Key (v3 auth) and paste above

⭐ OMDB API Key

For IMDB ratings and Rotten Tomatoes scores. Optional on deployed sites if env vars are already set on the host.

How to get your free API key:

  1. Go to omdbapi.com/apikey.aspx
  2. Select "FREE! (1,000 daily limit)"
  3. Enter your email and submit
  4. Check your email and click the verification link
  5. Copy your API key and paste it above

📦 Data Management

Export JSON for a full backup (episodes, TMDB cache) — use Import backup to restore. Export CSV for spreadsheets or the app’s Custom CSV import (main fields only).

Sharing the same list: Your deployed site must have SUPABASE_URL and a service role key in server env (e.g. Vercel). Send the Copy link URL from the app — it includes ?p=… (or ?list=shared for Watch together). They must open that link on the same live URL, not localhost on your PC. Open the app once yourself after Supabase is wired so your list uploads; then their refresh shows it.
